Functions

create (attributes, full)
Returns a new AssessmentStatement object. If full flag is not nil, the object will receive default children objects of each children class. This case, full must be passed to the method with a valid number.

Parameters:

  • attributes list of attributes to be initialized.
  • full numeric flag to indicate if the object will be created with filled children list.

Returns:

    new AssessmentStatement object created.
